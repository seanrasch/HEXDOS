HEXDOS 4.0 Emulator

Ohio Scientific Challenger 1P - Browser-based Historical Preservation

Based on HEXDOS by Steven P. Hendrix (1980-2015)

System Off
Press POWER ON to start the Ohio Scientific C1P emulator

System Information

CPU: MOS 6502 @ 1 MHz
Memory: 8K RAM
Display: 32x32 character text mode
ROM: OSI BASIC-in-ROM

Quick Start

  1. Click POWER ON to boot into OSI BASIC
  2. Click LOAD HEXDOS to load the HEXDOS operating system
  3. Use your keyboard to interact with the system
  4. Type HELP in HEXDOS for available commands

About HEXDOS

HEXDOS 4.0 is a disk operating system developed for the Ohio Scientific C1P computer in the early 1980s. This emulator preserves this historical software for educational and hobbyist purposes.

This is a non-commercial historical preservation project.

HEXDOS source code used under permission for non-commercial hobbyist purposes.
